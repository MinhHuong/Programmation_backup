Description générale

Suite au premier délivrable, ce deuxième consiste en l'intégration de deux maquettes (Application Android + Web)

Nous vous envoyons un rapport sour forme d'un web pour ajouter plus d'effets, et pour ne pas vous fatiguer en présentant un rapport de vingtaines pages (c'est juste des images, mais il est VRAIMENT long).

Site web

Cette intégration du web est réalisée de sorte de le faire marcher locallement, donc il n'y a aucune connexion au serveur. Des données sur chaque page sont temporaires, elles seront perdues une fois la page refraichie.

Cette intégration se compose de deux versions : des fichiers html, css, js pour le faire marcher en machine locale et des captures d'écran


*** SOURCE CODE ***

Des source code sont placé dans le répertoire /SiteWeb. Ce répertoire est organisé comme suit:

index.html (la page LOGIN)
| --- html/
| --- css/
| --- script/
| --- font/
| --- img/

Pour suivre l'enchainement des procédures pendant la gestion de vote, veuillez commencer par le fichier index.html.

Dans deux répertoires /css et /script, il existe 2 version : celle normale (.css/.js) et celle minifiée (.min.css/.min.js), pour but d'accélérer le page web le mieux possible.

Le site est déjà évalué sur Opquast : une erreur qui existe dans toutes les pages - le site web ne permet pas de CACHE (c'est notre intention).

Le site web est déjà testé sur plusieurs navigateurs, quelques problèmes notés:

  • Google Chrome: tout fonctionne bien
  • Firefox : il faut changer dans le menu "Options" pour bien voir des symboles, des popup Javascript
    • Content --> Pop-ups --> Uncheck "Block pop-up windows"
    • Content --> Fonts et Colors --> Advanced --> Check "Allow pages to choose their own fonts, instead of my selections above"
  • Internet Explorer : il faut confirmer “Allow popup content”
