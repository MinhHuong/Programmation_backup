D.  Các công việc nâng cao
  1. Nếu bạn có đường nối trực tiếp trên web, hãy đăng ký vào Writing HTML Alumni.
  2. Dave Siegel's Web Wonk, Tips for Writers and Designers, sẽ giúp bạn làm thế nào để tạo ra các trang web thật ấn tượng không giống như các trang bạn đã xem trước đây. Bạn cũng nên xem qua sách hay web site How to Create Killer Websites của ông ta.
  3. Bạn cũng nên đưa Yale C/AIM WWW Style Manual vào danh sách bookmark của bạn! Chúng tôi cũng thích xem Guide to Web Style của Sun Microsystems và NCD's HTML Design Guide.
  4. Nên thường xuyên xem các thông tin trong Kevin Werbach's Bare Bones Guide to HTML và webreference.com
